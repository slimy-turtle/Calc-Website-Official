Limits

Overview

Sometimes we are not able to get a direct answer on a problem but with limits, we are able to see what it should be as we get closer to one number.

Definition of Limits

There are certain instances where we need to find a specific point at a given number, giving the definition of the limit at said number. For example if you had the equation, lim{x → 5}(3x - 3). By plugging in the 5, you will have an answer of 12. This is a simple way to learn what the limit is at a given number.

Continuity

Sometimes you will be given a graph and asked, "Is this graph continuous at x → a?" (The a represents a number that we don't know yet). When looking at the graph on the right, you can see that the lines go to two different points at x = -2. From the left it goes to (-2,2) and from the right it goes to (-2,-1). Because they go to two different points at the same x value then if asked if the graph is continuous at x → -2, the answer would be no. If the question asked if the graph is continuous at x → 0, then the answer would be yes because it is continuous from both the left and right side.

Infinite Limits

Sometimes they will give you an equation where the limit is to infinity. When you have problems like that you often will end up needing to sub in the infinity or use a rule called L'Hospital's. For example, if given an equation of 1/∞, the answer would be undefined. But if we plug in numbers to the denominator, replacing the infinity, we will see that the equation is getting closer to 1/0.001. Therefore, using limits, we are able to determine that the limit of 1/x as x approaches infinity is 0.
